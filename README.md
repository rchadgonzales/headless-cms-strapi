# üöÄ Getting started with Strapi

Strapi comes with a full featured [Command Line Interface](https://docs.strapi.io/developer-docs/latest/developer-resources/cli/CLI.html) (CLI) which lets you scaffold and manage your project in seconds.

### `develop`

Start your Strapi application with autoReload enabled. [Learn more](https://docs.strapi.io/developer-docs/latest/developer-resources/cli/CLI.html#strapi-develop)

```
npm run develop
# or
yarn develop
```

### `start`

Start your Strapi application with autoReload disabled. [Learn more](https://docs.strapi.io/developer-docs/latest/developer-resources/cli/CLI.html#strapi-start)

```
npm run start
# or
yarn start
```

### `build`

Build your admin panel. [Learn more](https://docs.strapi.io/developer-docs/latest/developer-resources/cli/CLI.html#strapi-build)

```
npm run build
# or
yarn build
```

# Strapi.js Crash Course, Headless CMS

This is the project from the [YouTube crash course](https://www.youtube.com/watch?v=6FnwAbd2SDY). It includes the Strapi headless CMS (Content Management System) and create a REST API with JWT authentication all from the admin area of Strapi.

## Running Locally

**Back-end:**

1. `npm install`
1. `npm run develop`
1. App will run on port `1337`
1. Go to [http://localhost:1337/admin](http://localhost:1337/admin) and create a new user.

1. Don't forget to allow get request permissions for events.
